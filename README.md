# Codeathon 2K26 Backend

Node.js + Express backend for recipient loading and mail sending.

## Structure

- `data/` CSV recipient lists
- `logs/` runtime logs

## Setup

1. Copy `.env.example` to `.env`.
2. Fill in your Gmail and mail settings.
3. Install dependencies and run the server.

```powershell
npm install
npm run dev
```

The API runs on `http://localhost:5000`.
